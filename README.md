# HTML Headings Practice Project

A foundational web project built to demonstrate how semantic HTML headers structure a webpage. It provides a direct, visual comparison of text hierarchies from `<h1>` down to `<h6>`.

### 🛠️ Key Highlights

* **Structural Hierarchy:** Showcases the correct use of sizes from `<h1>` (most important) to `<h6>` (least important).
* **SEO Foundations:** Implements standard structural practices that help search engines understand page content layout.
* **Visual Scaling:** Displays how default browser font sizes change automatically based on the heading level tag used.
* **Clean Code:** Features minimal, clean markup designed for learning core frontend architecture.
